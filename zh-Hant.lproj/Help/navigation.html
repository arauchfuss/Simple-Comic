Navigating a Comic

Page Order

Page order affects navigation in an intuitive fashion. When viewing a comic laid out from left to right, navigating right moves to the next page. In a comic laid out right to left, navigating left moves to the next page.

Page order affects navigation even when pages are displayed singly.

Turning the Page

Mouse Navigation

  • Click on the left half of the page view to turn the page left. Click on the right half of the page view to turn the page right.
  • Use the left and right page turn toolbar items.
  • Use the scroll-wheel while page scaling is set to Fit to Window.

Keyboard Navigation

  • Command left/right arrow
  • Spacebar or the Page-up/down keys
  • Arrow keys

    • Scrolling still works as expected until the edge of a page is reached, at which point a page turn occurs. In Fit to Window scaling mode the page turn is instant.

Move Directly to a Page

There are three methods for doing this.

Progress Bar

At the bottom of the comic window there is a progress bar, click within it to move to any point in the comic.

Jump to Page command

Hitting enter or selecting Jump to Page from the Go menu will bring up a dialog box where you can type a page number. Hitting enter will move to the chosen page.
