Preferences

General

These are the default page layout and navigation settings used by any newly opened comic.

Layout and navigation settings can be over-ridden on a per-session basis.

Page Order

If you are reading western comics this should be set to Left to Right. Manga readers on the other hand should choose Right to Left.

Page Scaling

  • Original Size: Images are not scaled.
  • Window: Images are scaled to fit within the window.
  • Window width: The width of the window scales the displayed images. Works best with only one page being displayed.

Two Page Spread

Should pages be laid out as they were in print?

Background Color

Any space in a window not covered by a page will be this color.
